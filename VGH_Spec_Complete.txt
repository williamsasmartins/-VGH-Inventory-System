1. VISÃO GERAL
Nome: VGH Controle de Estoque
Objetivo: Sistema de controle de inventário com entrada de materiais por código.
Funcionalidade Principal: Digitar código → Auto-preenche nome/descrição
2. STACK TECNOLÓGICO
Frontend: React + TypeScript + Tailwind CSS
Backend: Supabase (PostgreSQL)
Export: SheetJS (xlsx)
Cores: Verde #16A34A | Alerta #DC2626
3. BANCO DE DADOS
Tabela: materials
id (UUID PK), code (TEXT UNIQUE), name (TEXT), category (TEXT), unit (TEXT), current_stock (INTEGER), min_stock_alert (INTEGER DEFAULT 5), created_at (TIMESTAMPTZ)
Tabela: transactions
id (UUID PK), material_id (UUID FK), type ('IN'|'OUT'), quantity (INTEGER), notes (TEXT), created_at (TIMESTAMPTZ)
TRIGGER: Após INSERT em transactions, atualiza current_stock automaticamente (+quantity para IN, -quantity para OUT)
4. TELAS
Dashboard
Cards de materiais por categoria. Alerta VERMELHO quando estoque < 5. Busca por código/nome.
Movimento (PRINCIPAL)
AUTO-COMPLETE: Usuário digita código → Sistema busca na tabela materials → Preenche nome automaticamente. Toggle IN/OUT, quantidade, notas opcionais.
Histórico
Lista de transações. Filtros por material/tipo/período. Botão Exportar Excel.
Configurações
Adicionar/editar materiais. Definir limite de alerta.
